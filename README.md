# Polar Heatmap Widget — Final Release

**Technology:** Qt 5.15 | C++17 | OpenGL 3.3 Core

## Build & Run
```bash
mkdir build && cd build
cmake ..
make -j4
./PolarHeatmapWidget
```

## All features implemented

| Feature |
|---------|
| OpenGL 3.3 Core Profile 
| VBO/VAO rendering
| Heatmap colour mapping 
| Range labels 
| Dynamic angular layout 
| Proper destructor 
| Qt texture
| Zoom / Pan / Reset
| Hover tooltip
| Sector selection 
| Live animation 
| Control panel
